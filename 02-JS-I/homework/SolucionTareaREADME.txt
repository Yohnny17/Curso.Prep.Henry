QUE SON VARIABLES?

sirve para guardar algo y poderlo recuperar de alguna forma.
a esa variable se le puede asignar un nombre y hacerla que pueda cambiar.
EJEMPLO:
var perez = "hola"

una de las variables son.

var
let
cons

variable var se usa para que una variable se ejecute en todo el codigo.

variable let se usa para que una variable se ejecuta en una parte de codigo en concreto.

variable cons se usa para que una variable se ejecute, pero no se puede cambiar su valor fijado.


QUE SON STRINGS?

son un tipo de datos la cual son cadenas de texto que van dentro de una variable y se escriben entre comillas ("").



QUE SON FUNCIONES

es agrupar codigo y despues se invoca para que se ejecute.

las funciones como su nombre lo dice, son funciones que son programadas para usarsen entre variables; las variables pueden ser de tipo string o de tipo number.

EJEMPLO:

function mifuncion() {
	
};

const otrafuncion = function () {
	
};






QUE SON NUMEROS BOLEANOS

los numeros booleanos son en otras balabras como decir 0 o 1 , "false" or "true" (falso o verdadero). y son palabras reservadas, osea que no se puede usar para declarar una variable.



Declaraciones `if

las declaraciones if se usan para dar una funcion si una o varias condiciones se cumplen.
